# 🌦️ Weather App

A simple and responsive Weather App built using HTML, CSS, and JavaScript that fetches real-time weather data using the OpenWeather API.

## 🚀 Live Demo
👉 https://weather-info-002.netlify.app

## 📌 Features
- Search weather by city name
- Real-time weather data
- Displays temperature and weather condition
- Responsive design (mobile-friendly)
- Clean and modern UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## 💡 How It Works
- User enters a city name
- JavaScript fetches data from API
- Weather details are displayed on screen
